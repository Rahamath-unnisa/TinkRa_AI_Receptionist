# 🤖 TinkRa AI Receptionist

An AI-powered virtual receptionist developed using **Python** and **CustomTkinter** for **TinkEdge Robotics Lab**. The application provides an interactive receptionist experience with voice interaction, staff authentication, visitor assistance, and a modern dashboard interface.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Rahamath-unnisa/TinkRa_AI_Receptionist.git
```

Go into the project

```bash
cd TinkRa_AI_Receptionist
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
python main.py
```
