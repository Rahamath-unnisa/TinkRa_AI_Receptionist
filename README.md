# 🤖 TinkRa AI Receptionist

An AI-powered virtual receptionist built with Python.

- Modern GUI
- Staff Authentication
- Visitor Support
- Knowledge Base
- Dashboard
- Chat Interface

## Technologies

- Python
- CustomTkinter
- Edge TTS
- SpeechRecognition
- Pillow
- Pygame

## Run

```bash
pip install -r requirements.txt
python main.py
```
